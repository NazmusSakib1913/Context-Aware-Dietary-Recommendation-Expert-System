# NutriSense AI: Context-Aware Dietary Recommendation Expert System

A user-friendly, intelligent dietary recommendation system that suggests healthy meal plans based on a user's personal information, health conditions, and dietary preferences. Built for a university Artificial Intelligence and Expert System course.

## Features
- User Registration & BMI Calculator
- Rule-based Expert System for personalized dietary filtering
- Daily meal recommendations (Breakfast, Lunch, Dinner, Snacks, Water)
- Visualizations using Plotly
- Weekly planner & Grocery List (Bonus)

## How to run
1. Install requirements: `pip install -r requirements.txt`
2. Run the application: `streamlit run app.py`
